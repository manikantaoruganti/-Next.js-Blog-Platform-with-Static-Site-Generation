# CSS Modules vs Tailwind CSS

When building React apps, styling is a big decision.

## CSS Modules

- Scoped locally by default.
- Standard CSS syntax.

## Tailwind CSS

- Utility-first framework.
- Rapid UI development.

**Which one to choose?** It depends on your team's preference.
